# Flutter Practice Project

A simple Flutter demo app that demonstrates registration, login, and profile flow.

## Overview

This project includes:
- A registration form with validation for name, email, and password.
- A login screen with a simulated loading state.
- Navigation through app screens using `Navigator`.
- Basic Material 3 theming with a clean card-style form layout.

## Folder structure

- `main.dart` — entry point of the app.
- `auth/registration_form.dart` — registration form screen with validation and loading state.
- `auth/login.dart` — login screen with simulated authentication.
- `screens/` — additional app screens such as biodata input, profile, and detail views.
- `widgets/` — reusable UI components like profile cards.

## Run the app

1. Install Flutter and set up the environment.
2. Open the project in your editor.
3. Run:
   ```bash
   flutter pub get
   flutter run
   ```

## Notes

This project is useful for learning Flutter form handling, navigation, and theming.
